System pre-registration and submission is now open.
  • Pre-Registration: submit an email to the organizers (Clark Barrett, Leonardo de Moura, Aaron Stump). The subject must be smt-comp pre-registration. The email must contain the system name, authors and their affiliations. It is not necessary to submit the system binaries or source code. Deadline: 15th of May.
  • System Submission: submit an email to the organizers (Clark Barrett, Leonardo de Moura, Aaron Stump). The subject must be smt-comp system submission. The email must contain a tarball with the system (sources or Linux executable). The tarball must include a README on compilation (if necessary) and installation; as well as a 1-2 page system description as explained in the rules relating to Entrants. Deadline: 1st of July.
