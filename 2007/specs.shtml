The competition will run on a cluster of 9 machines at Washington University in St. Louis. Some relevant specifications of these machines are:

  • 2.4 GHz AMD Opteron 250, configured for single core, 32-bit processing.
  • 1 MB cache.
  • 3.5 GB main memory.
  • Linux, kernel version 2.6.9-55.EL.
  • CentOS release 4.5 (Final)
  • glibc-2.3.4-2.36
  • gcc-3.4.6-8 (if a solver needs a higher version of GCC, this can be accomodated)
