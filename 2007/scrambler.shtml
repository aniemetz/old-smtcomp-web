Please note that this is a different scrambler than the one used last year. This scrambler is a new implementation by Aaron Stump in Java.

  • Source tarball

To build: tar xfz scrambler.tgz; cd scrambler; make
To use: java -jar scrambler.jar seed file.smt

[seed] is a seed for the pseudo-random number generator.

The scrambler performs the following functions:

  • it drops annotations;
  • it hides benchmark name and status;
  • it renames all variables; and
  • it randomly permutes arguments to AC logical connectives and theory function symbols.
