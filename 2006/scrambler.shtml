Usage: ./scrambler [seed] [file.smt]

[seed] is a seed for the pseudo-random number generator.

You must have sed (4.1.2 or newer) installed in your system to be able to use the formula scrambler.

If the benchmark is too big, the scrambler may run out of stack space and produce a segmentation fault. This problem can be avoided by executing the following command: ulimit -s unlimited

The formula scrambler was implemented in Scheme using the Bigloo compiler.
